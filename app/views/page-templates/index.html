Page templates

There are three main page templates that will be rolled out across the NHS website.

A main topic hub represents the broadest part of a topic area, giving users links to smaller sub-topics. A sub-topic hub is then a collection of all the content pages that are grouped within that section of the overarching topic.

The main topic hub is a grand-parent page, the sub-topic hub is a parent page, and a content page is a child page (or grandchild of the main topic hub).

Guidance for using page templates

More detailed, written explanations for how and when to use the page templates can be seen on confluence.

There is guidance for:

  • main topic hubs
  • sub-topic hubs

There is no specific guidance for the overall structure of a content page, but there is guidance for the sideways navigation component that should be used at the bottom of each content page that sits within a sub-topic hub.
