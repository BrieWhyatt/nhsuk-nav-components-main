Sideways navigation

The sideways navigation component allows users to navigate through a number of related content pages that sit beneath a sub-topic hub.

Example

More in GPs

  • How to register with a GP surgery
  • Registering with a GP surgery outside the area you live
  • GP appointments and bookings
  • Using an online form to contact your GP surgery
  • Video consultations
  • What to ask your doctor
  • Using online services
  • Online health and prescription services

When to use the sideways navigation

Users should be able to navigate sideways through children pages of a sub-topic hub.

For example, in the case of the Coronavirus (COVID-19) main topic hub on NHS.UK, Vaccinations is a sub-topic hub that has multiple child content pages. These content pages that sit beneath the Vaccinations sub-topic, have a sideways navigation component at the bottom of each page.

They should not be used on main hub pages, or on content pages that are not grouped under a single subject (a sub-topic hub).

How to use the sideways navigation

The sideways navigation should be placed at the bottom of a content page, after the date component, but before the footer. This allows users to read the content before giving them the ability to move on.

The component should contain an H2 heading of "More in ...". For the example of the Coronavirus Vaccinations sub-topic, the H2 heading would say "More in Vaccinations".

The heading should be followed by a list of hyperlinks that take the user to each of the other child pages within the sub-topic.

For the page the user is currently viewing, the hyperlink should be removed to make the text the same colour as the body text on the page. A black pipe should be used to the left-hand side of this text as way of emphasising where a user currently is.

Once a user has viewed a page using the sideways navigation, the link should be styled as a visited link.

Benefits of the sideways navigation

Research

During testing, NHS.UK's Nav Team found users were confident using the sideways navigation. It allowed them to move sideways through related content, without interrupting their ability to evaluate if they were in the correct place for the information they needed.

This component supports longer journeys on NHS.UK.
