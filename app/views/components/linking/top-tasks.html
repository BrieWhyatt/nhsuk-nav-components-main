Components

There are two main types of components that have been created to improve the information architecture on the NHS website.

They are linking components and navigation components.

Linking components

  • Main topic hub
  • Sub-topic hub
  • Content page

Guidance for linking components

More detailed, written explanations for how and when to use the page templates can be seen on confluence.

Currently, there is guidance for:

  • card with arrow
  • key links
  • top tasks

There is no specific guidance for the overall structure of a content page, but there is guidance for the sideways navigation component that should be used at the bottom of each content page that sits within a sub-topic hub.

Navigation components

  • Main topic hub
  • Sub-topic hub
  • Content page

Guidance for navigation components

More detailed, written explanations for how and when to use the page templates can be seen on confluence.

Currently, there is guidance for:

  • breadcrumbs
  • sideways navigation

There is no specific guidance for the overall structure of a content page, but there is guidance for the sideways navigation component that should be used at the bottom of each content page that sits within a sub-topic hub.
